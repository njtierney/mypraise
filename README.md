# praiseme

The goal of praiseme is to deliver to praise to people\!

## Installation

``` r
remotes::install_github("njtierney/mypraise")
```

## Example of the `praise()` function

The `praise()` function delivers praise\!

``` r
library(praiseme)
praise()
#> [1] "You're the best, Nick!"
```
